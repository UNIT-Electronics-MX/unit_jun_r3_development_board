# Software Guide

This folder contains the software quick start and the full C++ examples collection.

## Quick Start

### 1. Hardware Setup

1. **Connect to Computer**: Use a USB-C cable to connect UNIT JUN R3.
2. **Install Drivers**: Most operating systems recognize the board automatically.
3. **Verify Connection**: Confirm the power LED is on.

### 2. Development Environment (Arduino IDE)

1. Install [Arduino IDE](https://www.arduino.cc/en/software).
2. Install board support for ATmega328P if needed.
3. Select **Arduino Uno** as board type.
4. Install required libraries:
   - Adafruit NeoPixel

### 3. First Test Sketch

Use `cpp_example_00_neo_blink` as the first validation sketch:

- Path: `software/examples/cpp_examples/cpp_example_00_neo_blink/`
- Detailed documentation: `software/examples/cpp_examples/cpp_example_00_neo_blink/README.md`
